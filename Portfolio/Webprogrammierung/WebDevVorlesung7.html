Vorlesung 7 – Sitzung vom 5.12.2022 und 8.12.2022

Mit JavaScript können Entwickler dynamische Funktionen zu einer Webseite hinzufügen, indem sie Code schreiben, der im Webbrowser des Benutzers ausgeführt wird und den Inhalt und das Layout der Seite manipuliert. Beispielsweise kann ein Entwickler JavaScript verwenden, um ein Formular zu validieren, bevor es abgeschickt wird, um ein Bildkarussell zu animieren oder um ein Spiel zu erstellen, das im Browser läuft.
Dynamische Webseiten bieten eine ansprechende und interaktive Benutzererfahrung, da sie es den Benutzern ermöglichen, mit der Seite durch das DOM auf eine Weise zu interagieren, die bei statischen Seiten nicht möglich ist.

Verschiedene JavaScript Befehle

Verschachtelung von .getElementById()

Die Funktion .getElementById() ist eine Methode des Dokumentobjekts in JavaScript, mit der Sie ein Element auf einer Webseite anhand seines eindeutigen id-Attributs auswählen können. Um .getElementById() zu verschachteln, können Sie damit ein Element auf der Seite auswählen und es dann erneut verwenden, um ein untergeordnetes Element des gerade ausgewählten Elements auszuwählen. Hier ist ein Beispiel, wie Sie .getElementById() verschachteln können, um ein untergeordnetes Element auszuwählen: 


   <div id="parent">
      <div id="child">Child Element</div>
      </div>
      <script>
      // Select the parent element
      var parent = document.getElementById("parent");
      // Select the child element within the parent element
      var child = parent.getElementById("child");
   </script>

In diesem Beispiel wird die Funktion .getElementById() verwendet, um das übergeordnete Element auszuwählen, und dann erneut verwendet, um das untergeordnete Element innerhalb des übergeordneten Elements auszuwählen. Auf diese Weise können Sie bestimmte Elemente innerhalb des DOM (Document Object Model) anhand ihrer id-Attribute auswählen. Sie können .getElementById() auch mehrfach verschachteln, um tiefere Elemente innerhalb des DOM auszuwählen.

nextSibling/previousSibling

Die Eigenschaften nextSibling und previousSibling sind Knotenobjekte, die den Zugriff auf den nächsten bzw. vorherigen Geschwisterknoten eines bestimmten Knotens ermöglichen. Ein Geschwisterknoten ist ein Knoten, der sich auf der gleichen Ebene wie ein anderer Knoten im DOM-Baum befindet. Zum Beispiel im folgenden HTML: 


   <div id="parent">
      <div id="child1"></div>
      <div id="child2"></div>
      <div id="child3"></div>
   </div>

Die Elemente child1, child2 und child3 sind Geschwister, da sie alle Kinder desselben übergeordneten Elements (parent) sind. Um mit den Eigenschaften nextSibling und previousSibling auf die nächsten oder vorherigen Geschwister eines Knotens zuzugreifen, können man die folgende Syntax verwenden: 


   // Select the second child element
   var child2 = document.getElementById("child2");
   // Access the next sibling element (child3)
   var nextSibling = child2.nextSibling;
   // Access the previous sibling element (child1)
   var previousSibling = child2.previousSibling;

In diesem Beispiel wird die nextSibling-Eigenschaft verwendet, um auf das Element child3 zuzugreifen, und die previousSibling-Eigenschaft wird verwendet, um auf das Element child1 zuzugreifen. Man sollte beachten, dass die Eigenschaften nextSibling und previousSibling null zurückgibt, wenn es keinen nächsten bzw. vorherigen Geschwisterknoten gibt. Außerdem ist auch wichtig zu beachten, dass die Eigenschaften nextSibling und previousSibling den nächsten bzw. vorherigen Geschwisterknoten als rohes DOM-Knotenobjekt zurückgeben und nicht als umhülltes Elementobjekt, wie Sie es mit getElementById oder querySelector erhalten würden. Um auf die Eigenschaften des Geschwisterknotens zuzugreifen, müssen Sie DOM-Knoten-Methoden und -Eigenschaften verwenden, wie nodeName, nodeValue und innerHTML.

Document.wirte()

Die Methode document.write() ist eine Methode des document-Objekts in JavaScript, mit der man Inhalte in eine Webseite schreiben können. Sie kann verwendet werden, um neue Inhalte in die Seite einzufügen oder bestehende Inhalte zu überschreiben. Hier ein Beispiel, wie Sie die Methode document.write() verwenden können: 


   <script>
      document.write("Hello, World!");
   </script>

Dieser Code fügt die Zeichenfolge "Hello, World!" an der Stelle in die Seite ein, an der das Skript ausgeführt wird. Die Methode document.write() wird häufig in Verbindung mit Schleifen oder bedingten Anweisungen verwendet, um dynamische Inhalte für eine Webseite zu erzeugen. Es ist wichtig zu beachten, dass die Methode document.write() nicht verwendet werden sollte, um Inhalte in eine Seite einzufügen, nachdem diese fertig geladen ist, da dadurch die gesamte Seite überschrieben wird.

jQuery

jQuery ist eine schnelle, kleine und funktionsreiche JavaScript-Bibliothek, die das Manipulieren von HTML-Dokumenten, die Behandlung von Ereignissen und Animationen mit einer benutzerfreundlichen API, die in einer Vielzahl von Browsern funktioniert, erheblich vereinfacht. Mit einer Kombination aus Vielseitigkeit und Erweiterbarkeit hat jQuery die Art und Weise, wie Millionen von Menschen JavaScript schreiben, verändert. Hier sind einige der wichtigsten Funktionen von jQuery:

  • HTML/DOM-Manipulation: jQuery macht es einfach, HTML-Elemente auf einer Seite auszuwählen und zu manipulieren. Man kann zum Beispiel die Methode .css() verwenden, um den Stil eines Elements zu ändern, die Methode .html(), um seinen Inhalt zu ändern, und die Methode .attr(), um seine Attribute zu ändern.
  • Ereignisbehandlung: jQuery bietet eine einfache und bequeme Möglichkeit, Ereignisbehandler an Elemente zu binden. Man kann zum Beispiel die Methode .click() verwenden, um eine Funktion anzugeben, die ausgeführt wird, wenn ein Element angeklickt wird, oder die Methode .hover(), um Funktionen für Mauseintritts- und Mausaustrittsereignisse anzugeben.
  • Animation: jQuery verfügt über eine Vielzahl von Methoden zur Erstellung von Animationen, wie .fadeIn() und .slideUp(). Mit diesen Methoden kann man fließende Übergänge zwischen verschiedenen Zuständen erstellen und seiner Seite so auf einfache Weise interaktive Effekte hinzufügen.
  • Ajax: jQuery macht es einfach, HTTP-Anfragen zu senden und die Antworten mit der Methode $.ajax() oder ihren Kurzformen wie $.get() und $.post() zu verarbeiten. So kann man Daten von einem Server abrufen und eine Seite aktualisieren, ohne die Seite aktualisieren zu müssen.

    • Normalerweise lädt man jQuery auf den eigenen Webserver, man kann aber auch einen Link setzen und dann jQuery nutzen. Außerdem hat das Framework die größte aktive Entwicklergemeinde.
    Die Verwendung von jQuery hat mehrere Vorteile:

    1. Einfachheit: jQuery vereinfacht die JavaScript-Programmierung erheblich. Man kann komplexe Effekte mit wenigen Zeilen Code erreichen, im Gegensatz zum Schreiben von langem und komplexem JavaScript-Code.
    2. Browserübergreifende Kompatibilität: jQuery funktioniert in vielen modernen Browsern, darunter Internet Explorer, Firefox, Chrome, Safari und anderen. Das bedeutet, dass man sich keine Gedanken über das Testen Ihres Codes in verschiedenen Browsern oder das Schreiben von browserspezifischem Code machen muss.
    3. Leistung: jQuery wurde entwickelt, um schnell und effizient zu sein, und wurde für hohe Leistung optimiert. Es bietet auch Methoden, die helfen, die Anzahl der DOM-Manipulationen zu minimieren und das Risiko von Speicherlecks zu verringern.
    4. Große Gemeinschaft: jQuery hat eine große und aktive Gemeinschaft von Nutzern und Entwicklern, was bedeutet, dass man eine Fülle von Ressourcen und Unterstützung finden kann, um Probleme zu lösen oder mehr über die Bibliothek zu erfahren.
    5. Erweiterbarkeit: jQuery ist erweiterbar, d. h. man kann seine eigenen Plugins erstellen oder von anderen erstellte Plugins verwenden, um die Funktionalität zu erweitern. Das macht es einfach, neue Funktionen zu Ihren Projekten hinzuzufügen, ohne das Rad neu erfinden zu müssen.

    Hier einmal der Unterschied zwischen JavaScript und jQuery: JavaScript: 

                   
   // Get the element
   var element = document.getElementById("myElement");
   // Toggle its visibility
   if (element.style.display === "none") {
   element.style.display = "block";
   } else {
   element.style.display = "none";
   }

    jQuery: 

    
   // Toggle the visibility of the element with ID "myElement"
   $("#myElement").toggle();

    Wie man sehen kann, ist die jQuery-Version viel kürzer und leichter zu lesen, und sie abstrahiert viele der Details der DOM-Bearbeitung. jQuery bietet mehrere Methoden zum Hinzufügen, Entfernen und Manipulieren von Elementen im DOM (Document Object Model). Hier ist ein kurzer Überblick über einige der wichtigsten Methoden:

    • .append(): Fügt Inhalt am Ende eines Elements hinzu.
    • .prepend(): Fügt den Inhalt am Anfang eines Elements ein.
    • .remove(): Entfernt ein Element aus dem DOM.
    • .empty(): Entfernt den gesamten Inhalt aus einem Element.
    • .before(): Fügt den Inhalt vor einem Element ein.
    • .after(): Fügt den Inhalt nach einem Element ein.

    Hier ein Beispiel, wie Sie diese Methoden verwenden können:

    
   // Add a new list item to the end of the list
   $("ul").append("<li>Item 4</li>");
   // Add a new list item to the beginning of the list
   $("ul").prepend("<li>Item 0</li>");
   // Remove the second list item
   $("ul li:eq(1)").remove();
   // Remove all the content from the list
   $("ul").empty();
   // Insert content before the first list item
   $("ul li:first").before("<li>Item -1</li>");
   // Insert content after the last list item
   $("ul li:last").after("<li>Item 5</li>");

    Zuletzt ist es noch sehr wichtig, dass man kein JavaScript und jQuery mischt. Auch wenn es verlockend ist kann dies zu ungeplanten Komplikationen führen.
