Vorlesung 3 – Sitzung vom 27.10.2022

Zu Beginn der dritten Vorlesung haben wir wieder einen Exkurs zu HTML gemacht. Dabei haben wir uns auf das „HT“ (Hypertext) in HTML fokussiert. Hypertext steht für ein über Links verbundenes Netz aus Text-, Bild- und Dateneinheiten in dem sich Nutzer je nach Interesse frei bewegen können.

Hypertext

Um das zu erstellen haben wir den <a>-Tag kennengelernt, mit dem auf andere Seiten oder Dateneinheiten verlinkt werden kann. Das wichtigste Attribut des <a>-Elements ist das href=““ Attribut, das das Ziel der Verbindung angibt. 


   <a href="home.html">Home</a>

Eine verlinkte Seite wird normalerweise im aktiven Browserfenster angezeigt, solange nichts anderes angegeben ist. Standardmäßig werden Links wie folgt angezeigt:

  • Ein nicht besuchter Link ist unterstrichen und blau
  • Ein besuchter link ist unterstrichen und lila
  • Ein aktiver link ist unterstrichen und rot

Es kann nicht nur auf externe HTML Dateien verlinkt werden, sondern auch auf Dateien in der eigenen Websitestruktur. Wenn diese im gleichen Ordner liegt, ist dazu als Link der volle Name der Datei nötig, z.B. home.html.
Bei vielen Unterseiten verliert man allerdings so schnell den Überblick, dass es sich eher eignet eine Ordnerstruktur anzulegen. Das haben wir am Beispiel der Website KopfüBar ausprobiert. Dazuhaben wir die einzelnen Unterseiten in entsprechend benannte Ordner gepackt.

Um jetzt von bar.html auf getraenke.html zu verlinken, muss ein absoluter Dateipfad angegeben werden <a href="getraenke/getraenke.html">Getränke</a>. Will man nun von der Getränke-Seite wieder zurück auf die Startseite, muss auf den Ordner darüber verwiesen werden. Dazu werden zwei Punkte vorangestellt: <a href="../bar.html">Startseite</a>.

Variablen in JavaScript

Variablen sind Container, um Daten zu Speichern. Der Behälter ist ein logischer Speicherplatz mit einem unveränderlichem Namen und einem Wert, der je nach Deklaration verändert werden kann. Es gibt mehrere Wege Variablen in JavaScript zu deklarieren: var, let oder const. Das Schlüsselwort var wurde in JavaScript von 1995 bis 2015 benutzt. let und const wurden in 2015 hinzugefügt. Wenn der code auch in alten Browsern funktionieren soll, sollte man var benutzen.

Schlüsselwort Funktion
let Legt eine Variable im aktuellen Gültigkeitsbereich an, die nicht gleich deklariert werden muss. Für diesen Fall erhält sie den Wert undefined
var Deklariert eine Variable unabhängig von ihrem Scope (Gültigkeitsbereich)
const Legt eine Konstante an, die anfangs an einen Wert gebunden wird, welcher später nicht verändert werden kann (immutable)

Da JavaScript keine starke Typisierung aufweist, können in einer Variablen alle Arten von Werten gespeichert werden. Der Typ einer Variablen richtet sich daher implizit nach dem Typ des darin gespeicherten Wertes. Notwendige Typänderungen werden eigenständig im Hintergrund vorgenommen.
Man kann mithilfe von typeof bspw. in der Konsole den Datentyp einer Variablen ausgeben lassen: 


   const aNumber = 6;
   const aString = "Hallo";
   console.log(typeof aNumber);
   console.log(typeof aFloat);

Strings können auch verkettet werden: let aString = „Hal“ + „lo“
Neben normalen Variablen gibt es auch noch Arrays. Das sind spezielle Variablen, die mehr als einen Wert beinhalten können. Der einfachste Weg ein Array zu deklarieren ist literal Schreibweise: 


   const array_name = [item1, item2, …];

Man kann die Werte eines Arrays mit einer Indexnummer aufrufen:

				
   const cars = ["Saab", "Volvo", "BMW"];
   let car = cars[0];

Nach Erstellung können verschiedene Methoden mit dem Array benutzt werden, wie z.B. .push() und .pop() um Elemente am Ende hinzuzufügen oder wegzunehmen.

Kontrollstrukturen

Mit Kontrollstrukturen werden Programmabläufe gesteuert. Es sind vor allem Verzweigungen und Schleifen, mit denen im Programm auf bestimmte Bedingungen reagiert wird oder Programmschritte wiederholt werden. Man unterscheidet zwischen Verzweigungen und Schleifen. Mittels Verzweigungen ist es möglich, die Ausführungen einer oder mehrere Anweisungsblöcke von Bedingungen abhängig zu machen. Schleifen ermöglichen, einen Anweisungsblock wiederholt ausführen zu lassen.

Verzwigungen

In einer Verzweigung (bedingte Anweisung) wird ein Anweisungsblock nur durchlaufen, wenn eine Bedingung erfüllt ist. Alternativ kann beim Nichterfüllen der Bedingung ein zweiter Anweisungsblock durchlaufen werden. Innerhalb der else-Bedingung kann ein weiterer if-Block aufgerufen werden. 


   if (Bedingung) {
      //Anweisungsblock des True-Zweigs (wird ausgeführt, wenn Bedingung erfüllt ist)
      Anweisung;
      Anweisung;
   } else if (Bedingung) {
      //Anweisungsblock des False-Zweigs (wird ausgeführt, wenn Bedingung nicht erfüllt ist)
      Anweisung;
      Anweisung;
   }

Schleifen

Schleifen dienen dazu, bestimmte Anweisungen zu wiederholen ohne, dass diese dazu so oft notiert werden müssen. Dazu gibt es mehrere Arten von Schleifen sowie einige Zusätze, die innerhalb dieser Schleifen verwendet werden können. Die meistgebrauchten Schleifen sind z.B. die while-Schleife oder die for-Schleife.
